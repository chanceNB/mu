# Frontend

Vue 3 + Vite + TypeScript workbench for the AI Learning OS Phase 6 UI.

## Scripts

```bash
npm install
npm run dev
npm test -- --run
npm run build
```

## Scope

- Deterministic workbench UI for knowledge bases, documents, RAG chat, citations, learner profile, learning path, generated resources, agent trace, and assessment.
- No marketing landing page.
