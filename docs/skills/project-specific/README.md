## How to Add a New Skill

1. Complete a feature and run retrospective.
2. Identify reusable patterns.
3. Create a new `.md` file in this directory.
4. Register in `docs/skills/SKILL_REGISTRY.md`.
5. Reference from relevant Cursor rules if needed.
